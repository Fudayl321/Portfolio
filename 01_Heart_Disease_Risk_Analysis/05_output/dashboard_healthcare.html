Step-by-Step: From Raw Data to Dashboard

1

Data Source

Dataset obtained from the UCI Machine Learning Repository — Heart Disease dataset (Cleveland Clinic Foundation, 1988).
🔗 https://archive.ics.uci.edu/dataset/45/heart+disease
Kaggle mirror: kaggle.com/datasets/ronitf/heart-disease-uci
1,025 patient records · 14 clinical features · Binary classification target

2

Data Quality Issues Found

8 rows Impossible age values (999) — data entry error → removed
5 rows Negative cholesterol values (-1) — physically invalid → removed
13 nulls Missing blood pressure (trestbps) → median imputed
15 nulls Missing cholesterol (chol) → median imputed (right-skewed dist)
12 nulls Missing max heart rate (thalach) → median imputed

3

Cleaning Decisions — Justified

Median chosen over mean for blood pressure and cholesterol because both distributions are right-skewed (confirmed by IQR analysis). Rows with impossible values were dropped rather than imputed — 13 rows out of 1,025 is a 1.3% loss, acceptable without risking model contamination from fabricated data. Final dataset: 1,012 rows · 18 columns including 4 derived label fields for readability.

4

SQL Queries (SQLite)

Clean data loaded into SQLite. SQL used for aggregations and risk profiling before passing to the ML pipeline and visualisation layer.

5

Machine Learning — Random Forest Classifier

Random Forest trained on 80% of data (stratified split). 5-fold cross-validation used to assess generalisation. Feature importance extracted to explain model decisions. AUC: 0.949 · CV Mean AUC: 0.941

SQL Queries Used

Risk Profile by Age & Gender

SELECT age_group, sex_label, COUNT(*) AS total, SUM(target) AS disease_count, ROUND(AVG(target)*100, 1) AS disease_rate_pct, ROUND(AVG(chol), 1) AS avg_chol FROM heart GROUP BY age_group, sex_label ORDER BY age_group, sex_label

Chest Pain Type Analysis

SELECT cp_label, COUNT(*) AS total, SUM(target) AS disease_count, ROUND(AVG(target)*100,1) AS disease_rate_pct, ROUND(AVG(oldpeak),2) AS avg_st_depression FROM heart GROUP BY cp_label ORDER BY disease_rate_pct DESC
